# RF-Signal-Integrity-Analyzer
The RF Signal Integrity Analyzer is a MATLAB-based tool developed to study the effects of impedance mismatch and transmission line behavior on high-frequency signals. 
The primary goals of this project are:

To model transmission line reflections caused by impedance mismatch

To visualize how RF parameters change across frequency

To understand the relationship between reflection coefficient, VSWR, and return loss

To observe impedance transformation along transmission lines

To analyze reflections in both frequency and time domains
